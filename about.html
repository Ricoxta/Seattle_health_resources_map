About This Project

For our project we wanted to create a simple, easy to use online mapped center for resources that can be useful for housing challenged individuals in Seattle. We envisioned a map that included several types of resources such as healthcare, food banks, shelters and more. These types of resources would be differentiated with different symbology. Having all of these locations present on one map would be convenient and time saving when someone is in need of these resources and doesnâ€™t want to spend the time finding individual places on a more general mapping website. The map would be interactive in that you could click (or tap in the case of a mobile interface) and it would bring up useful information like exact street address, hours, and of course what category it falls under.
